He creado una aplicación que se ejecuta mediante:
-> node app

Una vez ejecutado, la aplicación dará instrucciones para
ejecutar el servidor y proporcionará información para
saber que URL contienen el JSON de Users y la de 
index.html, about.html y upload para subir el archivo

Para ejecutar el servidor usar:
-> node app server

Info:

->>> GET /user url is: http://localhost:8000/v1/users/ <<<-

->>> Upload url is: http://localhost:8000/index.html, http://localhost:8000/about.html and http://localhost:8000/upload <<<-
